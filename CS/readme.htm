Developer Express Example
Using DXDocking for WPF in accordance with Composite Application Guidelines

This example uses DXDocking (http://devexpress.com/DXDocking) to Prism (https://github.com/PrismLibrary/Prism) adapters to make it possible to display content from different modules in DXDocking panels.

You can find sample updates and versions for different programming languages here:
http://www.devexpress.com/example=E1926.
